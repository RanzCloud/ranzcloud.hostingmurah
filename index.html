Cara Order Via Discord

  1. Pilih paket hosting yang Anda inginkan dari daftar produk di atas.
  2. Klik tombol hijau Ini Order Via Discord untuk masuk store Discord nya.
  3. Jika sudah gabung di discrod, Kalian jangan lupa Ambil Rolle di Chenel Adventure Hosting
  4. Jika sudah, Kalian Scroll ke bawah sampai nemu Chenel Open Ticket, Kalian Create Ticket/Ambil Ticket Untuk Membeli Host.
  5. Jika sudah create Ticket/Ambil Ticket, Kalian Tunggu Konfirmasi Admin Setelah konfirmasi, lakukan pembayaran sesuai instruksi yang diberikan.

Cara Order Via WhatsApp

  1. Pilih paket hosting yang Anda inginkan dari daftar produk di atas.
  2. Klik tombol Order via WhatsApp pada paket tersebut.
  3. Anda akan diarahkan ke aplikasi WhatsApp dengan pesan pemesanan yang sudah terisi.
  4. Kirim pesan tersebut dan tunggu konfirmasi dari tim kami.
  5. Setelah konfirmasi, lakukan pembayaran sesuai instruksi yang diberikan.

Kontak Kami

Jika Anda memiliki pertanyaan, jangan ragu untuk menghubungi kami melalui WhatsApp atau Discord.
